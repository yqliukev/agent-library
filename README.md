# Agent Prompting MD Library

A general-purpose markdown library for agent prompting.

This repository stores reusable prompt templates that help guide AI agents through common workflows such as discovery, research, and planning.

## Repository structure
**Root**: General use prompts
- **Projects**: Designed to be used in conjunction through multiple prompts or by multiple agents in large spanning projects

## How to use
Just pick an md file and append it onto your prompt!
